## Installation

```bash
$ npm i
```





### PostgresSql: 
```bash
create table db : development
```






### Docker 
```bash
docker-compose up -d
```


### Init Prisma Orm
```java
$ npx prisma migrate dev
```


```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

> [!CAUTION]
> Running the app PORT 3333
